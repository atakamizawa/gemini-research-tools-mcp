# Privacy Policy - Gemini Research Tools Plugin

## Overview

This plugin connects to Google's Gemini API to provide research and search capabilities within Dify.

## Data Collection

### What data is sent to external services

1. **Google Gemini API**
   - Search queries and research topics you provide
   - URLs you specify for analysis
   - Format instructions and language preferences

### What data is NOT collected

- This plugin does not collect or store any personal information
- This plugin does not track usage analytics
- This plugin does not share data with third parties other than Google Gemini API

## Data Processing

- All data processing occurs through Google's Gemini API
- Research results and search responses are returned directly to Dify
- No data is cached or stored by this plugin

## Third-Party Services

This plugin uses the following third-party services:

### Google Gemini API
- **Purpose**: AI-powered research, search, and content analysis
- **Privacy Policy**: https://policies.google.com/privacy
- **Terms of Service**: https://ai.google.dev/terms

## User Consent

By using this plugin, you consent to:
- Sending your queries to Google Gemini API
- Google's processing of your data according to their privacy policy

## Data Retention

- This plugin does not retain any user data
- Data retention by Google Gemini API is governed by Google's policies

## Security

- API keys are stored securely within Dify's credential management system
- All communications with Google Gemini API use HTTPS encryption

## Contact

For questions about this privacy policy, please contact the plugin author.

## Changes to This Policy

This privacy policy may be updated from time to time. Changes will be reflected in the plugin's documentation.
